# Windows Presentation Foundation (WPF)


Opettele WPF ja XAML kieli. XAML on .Microsoftin html:lää vastaava muotoilukieli graafisten käyttöliittymien tekoon. Oheisessa video kokonaisuudessa on paljon muitakin ohjelmoinnissa tarvittavia tekniikoita. 
Jottei opettelu mene suoraan koodin kopioinniksi, kannttaa alkaa tekemään jotain saman tyylistä muttei ihan samaa ohjelmaa, jolloin joutuu miettimään miten opittua täytyy soveltaa omassa ohjelmassa.
